# 🚀 Project Workflow Manager

## ✨ Overview

**Project Workflow Manager** is specially crafted to highlight the contributions of every employee — whether they are Developers, Managers, or Team Leaders. This tool ensures that each individual’s effort is recognized and the entire project stays on track.

## 🎯 Purpose

This project is designed to:

- 📌 **Showcase Employee Contributions:** Bring visibility to each team member's role and progress.
- 🛠️ **Manage Workflow Efficiently:** Streamline the workflow and task assignment across different roles.
- ⏱️ **Ensure Deadline Compliance:** Help teams meet their project deadlines with better coordination and task tracking.

## 🧠 Key Features

- Role-based activity tracking  
- Centralized project dashboard  
- Deadline and milestone management  
- Status visualization for ongoing tasks

## 👥 Who Should Use This

- **Developers** looking to stay on top of assigned tasks  
- **Managers** aiming for efficient team and deadline management  
- **Team Leaders** who want a transparent view of project flow and collaboration  

## 🧰 Tech Stack
- Database: postgresql
- Language: Python, JavaScript, HTML, CSS
- Backend:  FastApi

## 📦 Installation

pip install -r requirements.txt


##  Steps To Run

- First run `generate_tables.py` file to make tables and insert one main super user.
- After that run `main.py` file to turn on Project.
- After that login into Super user and go to `Add User` tab and upload `employee_data_sample.xlsx` file.
- After adding neccesary employees you good to go.
